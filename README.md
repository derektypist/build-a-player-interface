# Build a Player Interface

The aim is to build a Player Interface using abstract classes.  Libraries used include abc and random.

## Technologies

Uses Python3.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` to run the code.
